# C# 期末遊戲專題《QDash》
## :book:背景故事：
在一次無法回頭的選擇後，方塊踏入了一個只剩下反應與本能的世界。

在這裡，沒有地圖，沒有提示，
前方只有不斷逼近的障礙，
以及名為「無盡深淵」的試煉。

你能做的，只有跳躍，或蹲下。
每一次遲疑，都是終結。

**《QDash》**
一款以極簡操作，挑戰人類反射極限的 2D 跑酷遊戲。
活得越久，世界就越殘酷。
## :closed_book:編譯/安裝指南：
### 開發環境
* Unity Version: 6000.2.15f1
* Language: C#
* 平台支援: PC
### 安裝指南
>1.下載本儲存庫（Repository）的壓縮檔，並將其解壓縮至本機資料夾。

>2.開啟 Unity Hub，進入專案管理畫面。

>3.點擊 「新增（Add）」→「從硬碟新增專案（Add project from disk）」。

>4.選擇剛才解壓縮完成的專案資料夾，並加入至 Unity Hub。

>5.專案開啟後，在 Project 視窗中找到 Scenes 資料夾，並開啟 SampleScene。
>* 若未立即看到，可於 Project 視窗的搜尋欄輸入「SampleScene」進行搜尋。

>6.點擊 Unity 視窗上方的 Play 按鈕，即可開始遊玩遊戲。

## :video_game:操作說明：
<kbd>左Shift鍵</kbd> -> 蹲下

<kbd>空白鍵</kbd> -> 跳躍
## :robot:AI工具協作：
**1.資產生成**

本專案於宣傳與展示素材製作階段，使用 AI 工具（ChatGPT 搭配圖像生成工具）協助產生遊戲宣傳封面，並應用於 itch.io 平台。生成過程中透過文字提示明確設定遊戲風格（灰階、簡約、方塊風）、畫面比例與標題配置，並經多次調整，使成果符合遊戲整體視覺需求。

**2.程式協作**

在遊戲開發過程中，障礙物生成與移動系統曾因語法與邏輯設計不當，導致移動異常與難度成長不穩定。透過 AI 工具分析既有程式結構與 Unity 2D 物理系統後，重新設計以玩家存活時間為依據的難度調整機制，並使用數學函式動態控制障礙物生成間隔與移動速度，使遊戲難度能隨時間平滑提升。
