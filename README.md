# QDash

一、資產生成：
本專案於宣傳與展示素材製作階段，使用AI工具（ChatGPT搭配圖像生成工具） 協助產生遊戲宣傳用封面圖，並用於itch.io平台的遊戲頁面。
在生成過程中，透過文字描述明確指定遊戲整體風格（灰階、簡約、方塊風）、畫面比例（1:1）、標題配置（QDash，Q字體較大）以及幾何裝飾元素，並多次調整提示內容，使生成結果符合遊戲實際視覺風格與使用需求。

二、程式協作：
在遊戲開發過程中，於實作障礙物生成與移動系統時，曾參考網路上的Unity 物件移動相關程式語法，但實際套用後出現物件移動異常，以及速度無法隨遊戲時間正確成長等問題。
因此透過AI工具協助分析既有程式碼結構與Unity 2D物理系統的使用方式，並重新設計障礙物的生成頻率與移動速度成長邏輯。最終改以玩家存活時間作為難度調整依據，使用數學函式動態計算障礙物生成間隔與移動速度，使遊戲難度能隨時間平滑提升，並成功解決原先語法無法正常運作的問題。
